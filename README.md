# 🚀 AI Revenue Agent System

An AI-powered multi-agent system that analyses business leads, creates sales strategies, and generates outreach messages automatically.

---

## 🔥 What This Project Does

This system simulates a real B2B sales workflow using 3 AI agents:

- 🧠 Lead Analyst → Evaluates lead quality, pain points, and buying signals  
- 📊 Sales Strategist → Builds a practical sales strategy  
- ✉️ Outreach Writer → Generates a ready-to-send message  

---

## 💡 Key Features

- Upload CSV/Excel lead datasets  
- Select any company from the dataset  
- Automatic multi-agent analysis  
- Decision-focused output (not just data)  
- Real business use case (sales & revenue optimisation)

---

## 🖥️ Demo Workflow

1. Upload a dataset  
2. Select a company  
3. Click "Run AI Agents"  
4. Get:
   - Lead Analysis  
   - Sales Strategy  
   - Outreach Message

## ⚙️ Setup Instructions

1. Clone repo:

git clone https://github.com/kamranafridi9220-prog/ai-revenue-agent-system.git

2. Install dependencies:

pip install -r requirements.txt

3. Add your OpenAI API key in `.env`:

OPENAI_API_KEY=your_api_key_here

4. Run app:

streamlit run app.py
